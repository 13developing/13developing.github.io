基础语法

即，使用单个字符串来描述、匹配一系列符合某个句法规则的字符串。

下面表示的是一串手机号码的正则表达式：

  • 符号 ^ 表示以…开始
  • 中间括号内的表示 或者， 即第二位数字为3~9 之间的任何一个数字
  • \d 表示的是可以为任意数字 ，后面跟上一个 { 9 } 表征重复的次数 ，即后面可以是任意的9个数字
  • $ 表示结束

jS中的使用方法

  • 利用RegExp对象创建 正则表达式
  • 利用字面量进行创建
  • test( ) 检测文本是否符合正则的规则 ，返回 true / false

常见使用句法

解释：

  • ^: 表示字符串的开头。

  • [a-zA-Z_]: 匹配一个字母（大小写不限）或下划线。

  • ```
    [a-zA-Z0-9_]{2,9}

解释：

  • ^: 表示字符串的开头。
  • \d: 匹配一个数字字符。
  • {n}: 匹配前面的模式（这里是数字）恰好 n 次。
  • -: 匹配一个短横线。
  • |: 表示或者。
  • $: 表示字符串的结尾。
