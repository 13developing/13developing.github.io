java ：构造代码块 和 静态代码块

以下是我对于二者区别浅薄的认识 ：

构造代码块：

  • {…}花括号包裹起来的代码我们称之为代码块或代码段。而只有一对花括号包裹起来的代码我们称之为构造代码块。

  • 构造代码块在构造方法之前先执行，构造代码块与构造方法一样都是在类被实例化的过程中被调用的。即每调用一次构造函数，都会在构造函数之前调用 —构造代码块–
