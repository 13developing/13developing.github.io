JSON的初识

JSON 是JS的对象标记法，是一种轻量级（Light-Meight)、基于文本的(Text-Based)、可读的(Human-Readable)格式。目前多数Web接口返回的数据都为json字符串。

JSON 的语法规则

JSON 和 XML 的对比

  • JSON 没有结束标签,长度更短,读写更快
  • JSON 能够直接被JavaScript解释器解析
  • JSON 可以使用数组

JSON 和 JS 对象的互转

  1. JSON –> JS 对象 JSON.parse() 方法

  1. JS对象 –> JSON JSON.stringify() 方法
